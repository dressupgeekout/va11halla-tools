VA-11 Hall-A Tools ChangeLog
============================

Version 0.1.0 
-------------

DATE??? 

- Revitalized the project.
- The code itself is rearranged to match conventional Ruby source layout,
  including a new namespace `Va11halla`.
- IFF parsing and resource extraction is now in its own class.
- STRGs are extracted to a file instead of spamming the standard output.
- Preliminary interpretation of the SHDR chunk, which will allow isolating
  shader code for GPUs.
- Preliminary interpretation of the TPAG chunk is in place.
- Preliminary interpretation of the CODE chunk is in place.
